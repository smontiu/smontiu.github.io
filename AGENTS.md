# AGENTS

This repository contains a personal website for Santiago Montiu.

## Purpose

- Document the use of AI agents or Copilot in this workspace.
- Provide a simple reference for contributors on how to interact with the site.

## Notes

- No custom agent configuration is currently defined in the repository.
- The website is built from static HTML, CSS, and assets.
- Changes to this site can be made directly in `index.html` and related asset files.
- When any code change is made, update the `<lastmod>` date in `sitemap.xml`.
- If new routes or pages are added to the site, add corresponding `<url>` entries to `sitemap.xml`.

## Useful context

- The homepage is `index.html`.
- The sitemap is `sitemap.xml`.
- Assets are stored under `assets/` and `img/`.
